About

What is Origami?

Origami is an easy to use text editor and UI designer for creating PWAs, or Progessive Web Applications. It requires no previous knowledge of programming, but supports over 40 programming languages. It is currently very basic, but better features are planned.

How does it work?

The desktop edition of Origami runs on a framework known as Electron. Electron is based on the Chromium browser and allows applications to run on nearly every platform. It uses Ace, an embeddable code editor written in JavaScript, to handle keybindings and text editing. On the Chrome OS and Android platforms (Coming Soon!), Origami uses the respective platforms WebView component to handle the application.
